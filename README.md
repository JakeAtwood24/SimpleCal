# 🥦 SimpleCal

An ad-free, user-friendly, and all-encompassing daily calorie tracker.

</div>

---

## Description

SimpleCal was created in response to my usual calorie tracker app becoming an ad machine, constantly bombarding me with ads for every food I logged. I figured I could code a new calorie tracker to be better, cleaner, and easier to use than my ex-app, while removing all of the ad/paid elements.

## Feature List

### Current Features

- Information page that asks for gender, weight, age, average activity, and desired speed of weight loss.
- A home page that opens if the user has already filled out their information at one point.
- A calorie wheel that shows how much of the daily caloie total they have consumed.

### Planned Features

- Fully self-designed images and graphics to make it a pleasant user experience. (Including multiple themes?)
- Fully functioning streak feature that has 3 different states (day not logged, day logged and over budget, day logged and under) to incentivize both logging on and staying under your daily limit
- Graph to show percentage distribution of the main macronutrients you consumed in a day. (What % fat, protein, and carbs)
- Possible recipe/meal saving component
- Weight loss plan tracking and graph to adjust total daily calorie count over time.
- Use of an API to find existing prodcuts macronutrient information online for easy logging.
- Possible AI integration to determine calories in a meal from a picture.
- Possible expansion into a full blown, all encompassing health app

## Limitations/Possible Improvements:

As of now the calorie tracking portion is incrdibly underwhelming, and the UI doesn't look particularly good. I really want to nail the aesthetics, so I hope to design all of the graphics and images for the app myself. Additionally, I need to fix some of the navigation links and other performance issues as well.
